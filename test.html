📄 FU角色卡 - 测试工具

测试 Excel 解析 + 数据存储 + 全屏卡片

📂 步骤1：解析 Excel

选择你的 枭熊卡.xlsx 文件，然后点击解析。

解析后的数据会显示在下方，同时也会打印到浏览器控制台 (F12)
等待解析...

💾 步骤2：保存到仓库

解析成功后，给角色取一个 ID，然后保存。

💡 点击保存后，数据会存入浏览器的 localStorage，刷新页面也不会丢失。

📂 步骤3：读取 / 显示 / 删除

输入 ID，可以读取数据、显示全屏卡片，或删除。

点击后会在屏幕中央弹出角色卡，可以拖拽和修改数值
等待读取...

📖 使用流程：

  1. 步骤1：选择 Excel 文件，点击解析。确认下方显示的数据正确。
  2. 步骤2：输入一个 ID（比如角色名），点击保存。
  3. 步骤3：输入同样的 ID，点击 "显示全屏卡片"，看看漂亮的大卡片！
  4. 点击卡片上的数字（HP、MP等）可以直接修改，数据会自动保存。
  5. 拖拽卡片头部可以移动位置。

所有数据都存储在浏览器的 localStorage 中，可以随时打开 F12 查看。
